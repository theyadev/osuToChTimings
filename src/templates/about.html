About osu! to Clone Hero Converter

This web application is an online interface for the osuToCH tool, which converts timing points (BPM changes and time signatures) from osu! beatmap files to Clone Hero chart format.

Features

  • Direct download of osu! beatmaps from the official site
  • Automatic extraction of timing data from .osu files
  • Precise conversion of osu! timing points to Clone Hero's tick-based format
  • Supports multiple BPM changes and time signatures
  • Generates properly formatted Clone Hero SyncTrack sections

How It Works

  1. Downloading: The application downloads the beatmap from the osu! website.
  2. Extraction: It extracts the .osu file from the downloaded beatmap package.
  3. Parsing: The application parses the timing points from the [TimingPoints] section.
  4. Conversion: osu! timing points (in milliseconds) are converted to Clone Hero's tick-based system using a formula that ensures precise alignment.
  5. Output: The application generates a properly formatted SyncTrack section ready for use in Clone Hero charts.

Technical Details

osu! stores timing information in milliseconds, while Clone Hero uses a tick-based system (192 ticks per beat by default). The conversion process calculates the exact tick positions based on elapsed time and BPM changes to ensure accurate timing.

The formula used for conversion is: ticks = elapsed_minutes * BPM * tick_rate
