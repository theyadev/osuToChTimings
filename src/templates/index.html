.osu to .chart Timing Converter

Convert timing points (BPM changes and time signatures) from any osu! beatmap to; Moonscraper, Clone Hero, YARG, etc...; .chart format. Simply enter the URL of an osu! beatmap below.

Example: https://osu.ppy.sh/beatmapsets/410162#osu/890190

How It Works

  1. Enter the URL of an osu! beatmap set
  2. The application will download the beatmap and extract the .osu file and audio
  3. Timing points and metadata are extracted and converted to .chart format
  4. Download the complete .chart file and audio ready to use in your editor

Important Notes

  • For accurate timing conversion, use the audio file downloaded from this tool
  • The generated chart uses 192 ticks per beat (Resolution) - this is the standard in Moonscraper; other editors may require manual adjustment
  • Beatmaps with negative timing points are supported, but may not be 100% accurate
  • For best results, use ranked, qualified, or loved beatmaps as they have the most accurate timing
